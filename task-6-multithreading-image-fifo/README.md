# Multithreading Image FIFO

## Описание
* Реализован потокобезопасный буфер (FIFO) для асинхронной передачи тяжелых блоков данных (например кадров изображения) между потоками-производителями (Producer) и потоками-потребителями (Consumer).

## Тестирование
* Код покрыт Unit-тестами на фреймворке Google Test
