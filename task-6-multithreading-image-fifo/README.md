# Multithreading Image FIFO

## Описание
* Реализован потокобезопасный буфер (FIFO) для асинхронной передачи тяжелых блоков данных (например кадров изображения) между потоками-производителями (Producer) и потоками-потребителями (Consumer).

## Тестирование
* Код покрыт Unit-тестами на фреймворке Google Test

## Запуск
Для сборки проекта нужно выполнить:
1. В корневой папки этой задачи разместить папку с исходным кодом googletest.
2. В корневой папки этой задачи создать папку build и перейти в неё: `mkdir build && cd build`.
3. Выполнить `cmake ..`.
4. Выполнить `cmake --build .`.
5. Запустить тесты `./bin/image_fifo`.
