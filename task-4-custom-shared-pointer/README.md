# Custom Shared Pointer

## Описание
* Реализован собственный умный указатель SharedPTR, а также функции make_shared. Под капотом указатель использует отдельный управляющий блок (ControlBlock), который хранит счетчик ссылок, указатель на объект и deleter. (Это очень удобно для копирования и перемещения указателей).
* Реализован механизм монолитного выделения памяти (один блок памяти для ControlBlock и самого объекта).
* Реализована поддержка типов-массивов (SharedPTR<T[]>)
* Поддержка пользовательских стратегий удаления объектов через TDeleter (для управления ресурсами, например открытие/закрытие файлов или TCP соединений)

## Тестирование
Код покрыт Unit-тестами на фреймворке Google Test, покрывающими все возможные варианты использования

## Запуск
Для сборки проекта нужно выполнить:
1. В корневой папки этой задачи разместить папку с исходным кодом googletest.
2. В корневой папки этой задачи создать папку build и перейти в неё: `mkdir build && cd build`.
3. Выполнить `cmake ..`.
4. Выполнить `cmake --build .`.
5. Запустить тесты `./bin/shared_ptr`.
